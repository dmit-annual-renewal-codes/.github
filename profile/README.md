## 先搞清楚一件事：什么叫"循环折扣"

DMIT 的优惠码分两种。

一种是**一次性折扣**：首单打折，续费恢复原价。便宜了一次，然后就没了。

另一种是**循环折扣（Recurring）**：只要你的账号绑定了这个码，续费时系统会自动按折扣价结算，不需要每次重新输入，永久生效直到你主动取消。

对于想长期用 DMIT 的人来说，循环折扣才是真正有价值的东西。毕竟 VPS 这东西，不是买一次就完了，年年都得续费。

**核心逻辑就一句话：能拿到循环折扣码的时候，别犹豫，先买。**

## 几个常见的踩坑场景

**优惠码输进去没反应？** 大概率是三个原因：付款周期不对（要选季付或年付）、套餐档位不符合（部分码要求 STARTER 以上）、活动已结束。

**现有套餐能不能换码？** 理论上不能直接更换。DMIT 系统不支持在续费时临时追加优惠码——循环码只在**首次下单时**绑定，之后自动沿用。如果你之前没有输入码，想补救，只有一种方法：退款重新购买（3 天内 + 流量 ≤30GB 可全额退款），这次下单时把码带上。

**账户金是什么？** 部分优惠码除了即时折扣外，还会按月分期返还"账户余额"到你账户里，可用于续费抵扣，但无法提现。换句话说，这是 DMIT 锁住你继续用的方式，对长期用户来说其实挺划算。

## 什么时候买最划算

DMIT 不是那种天天搞活动的商家，节日促销是目前最主要的大折扣来源，过去几年基本是圣诞/新年、黑五前后会有比较大的循环码出来，平时则是偶发的产品补货或线路调整时附带少量专属码。

**最优购买时机排序：**

1. 节日促销期间 + 年付 + 循环码叠加 = 综合折扣最大
2. 平时选年付结算，自带比月付多出的折扣
3. 关注 DMIT 官方公告，产品线调整时往往有专属码

有一点值得特别说一下：DMIT 的历史记录里，"买二送一"这种活动出现过不止一次，尤其是 Pro 系列。如果你同时有多台机器的需求，等这种活动出来一次性买，是真的省很多。

5. 点 "Validate Code"——如果折扣生效，价格会立刻变化
6. 确认折扣正确后完成结账

支持支付宝、微信、PayPal，国内用户付款很方便。
